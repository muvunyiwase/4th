Common Issues & Solutions

1. "Network error" or "Failed to fetch"

  • Backend not running: Run cd backend && npm run simple
  • Wrong port: Check console for backend port
  • CORS issue: Backend needs proper CORS headers

2. Backend running but not accessible

  • Firewall blocking port 3000
  • Another service using port 3000
  • Backend crashed - check terminal for errors

3. Form works but no email

  • Email not configured (test mode only)
  • Gmail credentials missing or incorrect
  • Check backend logs for email errors
